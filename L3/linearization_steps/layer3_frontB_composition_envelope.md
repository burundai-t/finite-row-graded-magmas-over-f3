# Layer 3 Front B — structural classification of the composition envelope

**Version:** Layer 3 v1.2 / Front B.  
**Object:** the selected PAB magma \((M,\cdot)\) with \(M=S\times S\), \(S=\mathbb F_3\).  
**Goal:** replace the raw closure statement
\[
|\operatorname{Mon}\langle L_x\rangle|=51,\qquad
\dim\mathcal E=27,\qquad
\dim W_r=9
\]
by a structural classification.

Throughout this note, \(\operatorname{Mon}^+\langle L_x\rangle\) means the **positive** composition monoid/semigroup generated by the nine left translations. No external identity map is adjoined. With an externally adjoined identity, the size would be \(52\); the Layer 3 envelope discussed here is the positive generated object of size \(51\).

---

## 1. Left translations and row fibers

Write
\[
F_r=\operatorname{span}\{e_{(r,c)}:c\in S\}.
\]

For \(x=(r,a)\), the left translation \(L_{r,a}\) has target row \(r\):
\[
L_{r,a}(k[M])\subseteq F_r.
\]

On the source fiber \(F_s\), it acts as follows.

If \(s\ne r\), then PAB cross-row multiplication gives
\[
(r,a)\cdot(s,c)=(r,s),
\]
so \(L_{r,a}|_{F_s}\) is the constant column map
\[
K_s:c\mapsto s.
\]

If \(s=r\), define the same-row block
\[
B_{r,a}:S\to S
\]
by
\[
B_{r,a}(c)=
\begin{cases}
r, & c=a,\\
\overline{ac}, & c\ne a.
\end{cases}
\]
Then
\[
L_{r,a}|_{F_r}=B_{r,a}.
\]

So every generator has the block form
\[
L_{r,a}
=
(K_0,\ldots,K_{r-1},B_{r,a},K_{r+1},\ldots,K_2)
\]
with output in \(F_r\).

---

## 2. The same-row transformation semigroup \(\mathcal B_r\)

Let
\[
\mathcal B_r=\langle B_{r,0},B_{r,1},B_{r,2}\rangle
\]
inside the full transformation semigroup of \(S\).

Define \(\tau_r\) to be the transposition fixing \(r\) and swapping the two elements of \(S\setminus\{r\}\).

For \(p\ne q\), define
\[
N^{(r)}_{p,q}(r)=p,
\qquad
N^{(r)}_{p,q}(c)=q\quad(c\ne r).
\]

Then
\[
\boxed{
\mathcal B_r
=
\{\operatorname{id},\tau_r\}
\cup
\{N^{(r)}_{p,q}:p\ne q,\ p=r\text{ or }q=r\}.
}
\]

Therefore
\[
\boxed{|\mathcal B_r|=6.}
\]

For \(r=0\), the six maps, written as triples \(c=0,1,2\mapsto U(c)\), are:
\[
011,\quad 012,\quad 021,\quad 022,\quad 100,\quad 200.
\]

Here \(012=\operatorname{id}\), \(021=\tau_0\), and the remaining four are rank-two maps constant on \(S\setminus\{0\}\).

The same table for \(r=1,2\) is obtained by relabeling.

## 3. Normal forms for positive composition words

Let \(K_{r,t}\) be the global constant map
\[
K_{r,t}(s,c)=(r,t)
\qquad\forall(s,c)\in M.
\]

For \(U\in\mathcal B_r\) and \(a\in S\), define the row-stay map
\[
T_{r,U,a}:M\to F_r
\]
by
\[
T_{r,U,a}(s,c)=
\begin{cases}
(r,U(B_{r,a}(c))), & s=r,\\
(r,U(s)), & s\ne r.
\end{cases}
\]

Then every positive composition word in the \(L_x\)'s is exactly one of the following two forms:
\[
\boxed{K_{r,t}}
\]
or
\[
\boxed{T_{r,U,a}}.
\]

### Proof

The initial generator is
\[
L_{r,a}=T_{r,\operatorname{id},a}.
\]

Assume a word \(T\) has target row \(s\).

If a new left factor \(L_{r,a}\) has \(r\ne s\), then the entire image of \(T\) lies in \(F_s\), and \(L_{r,a}\) is constant on \(F_s\):
\[
L_{r,a}\circ T=K_{r,s}.
\]

If \(r=s\), then composition remains in the same target row:
\[
L_{r,a}\circ T_{r,U,b}=T_{r,B_{r,a}\circ U,b},
\]
and
\[
L_{r,a}\circ K_{r,t}=K_{r,B_{r,a}(t)}.
\]
Since \(\mathcal B_r\) is closed under composition, the normal-form class is closed. This proves the classification by induction on word length.

## 4. Counting the \(17\) elements in each \(W_r\)

The constant maps contribute
\[
\boxed{3}
\]
elements \(K_{r,0},K_{r,1},K_{r,2}\).

For row-stay maps \(T_{r,U,a}\):

- if \(U=\operatorname{id}\) or \(U=\tau_r\), then \(U\) is injective, and the three choices \(a=0,1,2\) give three distinct row-stay maps;
- if \(U=N^{(r)}_{p,q}\), then \(U\) is constant on \(S\setminus\{r\}\). Hence \(T_{r,U,a}\) depends only on whether \(a=r\) or \(a\ne r\), so the three choices of \(a\) collapse to two distinct maps.

There are two bijective \(U\)'s and four rank-two \(U\)'s, hence
\[
\#\{T_{r,U,a}\}
=
2\cdot3+4\cdot2
=
14.
\]

Adding the three global constants gives
\[
\boxed{|W_r\cap\operatorname{Mon}^+\langle L_x\rangle|=14+3=17.}
\]

Since the target rows \(F_0,F_1,F_2\) are disjoint,
\[
\boxed{
|\operatorname{Mon}^+\langle L_x\rangle|
=
3\cdot17
=
51.
}
\]

This proves the former verifier-only monoid count structurally.

## 5. Linear envelope and row-target components

Define
\[
\mathcal E=\operatorname{span}\operatorname{Mon}^+\langle L_x\rangle.
\]

For each \(r\),
\[
W_r=\operatorname{span}\{T\in\operatorname{Mon}^+\langle L_x\rangle:\operatorname{Im}T\subseteq F_r\}.
\]

Because the output row supports are disjoint,
\[
\mathcal E=W_0\oplus W_1\oplus W_2.
\]

The exact row rank audit gives
\[
\boxed{\dim W_r=9\qquad(r=0,1,2).}
\]

For \(r=0\), one explicit \(9\)-element basis is given by the following triples. A triple
\[
\phi_0|\phi_1|\phi_2
\]
means that the restriction to source fiber \(F_s\) is \(c\mapsto\phi_s(c)\).

| basis index | triple |
|---:|---|
| 0 | \(000|000|000\) |
| 1 | \(011|000|000\) |
| 2 | \(011|111|111\) |
| 3 | \(012|222|111\) |
| 4 | \(021|111|222\) |
| 5 | \(022|000|000\) |
| 6 | \(100|000|000\) |
| 7 | \(100|111|222\) |
| 8 | \(200|000|000\) |

The corresponding \(9\times27\) flattened matrix has rank \(9\). The basis tables for \(r=1,2\) are produced by the verifier and agree by relabeling.

Therefore
\[
\boxed{\dim\mathcal E=9+9+9=27.}
\]

---

## 6. Product law

If \(A\in W_r\) and \(B\in W_s\), then
\[
B(k[M])\subseteq F_s.
\]

But \(A\) maps every source fiber into \(F_r\). Hence
\[
AB(k[M])=A(B(k[M]))\subseteq A(F_s)\subseteq F_r.
\]

Therefore
\[
\boxed{W_rW_s\subseteq W_r.}
\]

This is the operator-envelope version of AX0 row-grading: in composition, the left target row absorbs the right target row.

---

## 7. Final Front B theorem

\[
\boxed{
\operatorname{Mon}^+\langle L_x\rangle
=
\bigsqcup_{r\in S}
\left(
\{K_{r,0},K_{r,1},K_{r,2}\}
\sqcup
\{T_{r,U,a}:U\in\mathcal B_r,\ a\in S\}_{\mathrm{distinct}}
\right).
}
\]

For each \(r\):
\[
\boxed{
|\mathcal B_r|=6,\qquad
\#\{T_{r,U,a}\}_{distinct}=14,\qquad
|W_r\cap\operatorname{Mon}^+\langle L_x\rangle|=17.
}
\]

Globally:
\[
\boxed{
|\operatorname{Mon}^+\langle L_x\rangle|=51,
\qquad
\mathcal E=W_0\oplus W_1\oplus W_2,
\qquad
\dim W_r=9,
\qquad
\dim\mathcal E=27,
\qquad
W_rW_s\subseteq W_r.
}
\]

Thus Front L3-B is closed at the same standard as Front A: the finite computation has been replaced by a compact normal-form theorem plus verifier audit tables.
