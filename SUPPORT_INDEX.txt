PAB PREPRINT SUPPORT INDEX
==========================

Purpose
-------
This file maps the current manuscript to the repository support layers and
reviewer-facing verifier commands. It is a navigation index, not a second
mathematical manuscript.

Canonical manuscript
--------------------
Mathematical source:
  paper/main.tex

Compiled manuscript:
  paper/main.pdf

Build command:
  cd paper
  lualatex -interaction=nonstopmode -halt-on-error main.tex
  lualatex -interaction=nonstopmode -halt-on-error main.tex
  lualatex -interaction=nonstopmode -halt-on-error main.tex
  cd ..

The layer notes inside L1/, L2/, and L3/ are frozen reference notes. For theorem
wording, use paper/main.tex. For support paths and commands, use this index and
README.md.

Default reviewer-facing checks
------------------------------
Finite L1/L2/L3 quick audit:
  tools/run_quick_checks.sh | tee VERIFY_LOGS/quick_checks.log

mathcal_H default artifact audit:
  tools/run_mathcal_H_audit.sh | tee VERIFY_LOGS/mathcal_H_audit.log

mathcal_H audit including the heavier H2 focused artifact audit:
  tools/run_mathcal_H_audit.sh --include-h2-full | tee VERIFY_LOGS/mathcal_H_audit_with_h2_full.log

The default mathcal_H audit uses the light H2 coverage/consistency check. It
does not rerun the SMT solver. The focused H2 artifact audit is available via
--include-h2-full, and full recomputation commands are listed below and in
README.md.

Full recomputation routes
-------------------------
Full recomputation commands can be long. They write to mathcal_H/runs/, which is
ignored by .gitignore. Append --dry-run for a no-work preflight if supported by
the runner.

H1 full recomputation:
  cd mathcal_H
  PYTHONDONTWRITEBYTECODE=1 python3 h1/run_h1_evaluator.py \
    --start-index 0 \
    --end-index 0 \
    --run-dir runs/h1_full \
    --force \
    --self-test
  cd ..

H2 full pure-frontier recomputation:
  cd mathcal_H
  PYTHONDONTWRITEBYTECODE=1 python3 h2/run_h2_frontier.py \
    --depth 9 \
    --start-offset 0 \
    --end-offset 0 \
    --run-dir runs/h2_full \
    --force
  cd ..

H4 full recomputation:
  cd mathcal_H
  PYTHONDONTWRITEBYTECODE=1 python3 h4/run_h4_classifier.py \
    --start-index 0 \
    --end-index 0 \
    --run-dir runs/h4_full \
    --force \
    --self-test
  cd ..

Spot recomputation routes
-------------------------
Spot recomputation checks arbitrary bounded segments. It is a reviewer-facing
sanity route, not a replacement for the accepted full artifacts.

H1 spot recomputation:
  cd mathcal_H
  PYTHONDONTWRITEBYTECODE=1 python3 h1/run_h1_evaluator.py \
    --start-index 123456 \
    --end-index 124456 \
    --run-dir runs/h1_segment_123456_124456 \
    --force \
    --self-test
  cd ..

Integrity manifest
------------------
Release integrity manifest:
  MANIFEST.sha256

Generator:
  tools/make_manifest.sh

Verification command:
  sha256sum -c MANIFEST.sha256

The manifest excludes .git/, MANIFEST.sha256 itself, Python cache files,
.DS_Store, and local recomputation outputs under runs/ or mathcal_H/runs/.
